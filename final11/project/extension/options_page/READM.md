**Призначення:**  
Дозволяє користувачу налаштовувати роботу розширення.  
**Основні функції:**
- Включення/виключення автоматичного аналізу на певних сайтах.
- Вибір мови аналізу.
- Очистка локальної історії оцінок.
- Налаштування сповіщень (звук, push).  

**Фреймворки / бібліотеки:**  
- HTML, CSS, JS
- **Bootstrap** або **TailwindCSS** для зручного UI
