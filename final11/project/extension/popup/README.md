**Призначення:**  
Інтерфейс користувача, який відображає результат аналізу.  
**Основні функції:**
- Показ результату аналізу: "Фейк / Не фейк".
- Відображення відсотка довіри.
- Може показувати коротке пояснення, чому стаття вважається фейком.
- Доступ до налаштувань через кнопку "Налаштування".  

**Фреймворки / бібліотеки:**  
- HTML, CSS, JavaScript
- Можливе використання **Bootstrap** або **TailwindCSS** для швидкого стилювання UI
